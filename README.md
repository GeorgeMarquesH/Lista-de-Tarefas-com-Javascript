# ✅ Lista de Tarefas com JavaScript

## 📌 Descrição
Projeto em **HTML, CSS e JavaScript** que implementa uma lista de tarefas interativa.  
Permite adicionar, editar, remover e limpar tarefas de forma simples e prática.

---

## 🚀 Etapas principais
1. Estruturação da página com HTML5.
2. Estilização da interface com CSS3.
3. Implementação da lógica em JavaScript:
   - Adicionar tarefas.
   - Editar tarefas existentes.
   - Remover tarefas individualmente.
   - Limpar toda a lista.
   - Exibir mensagens de erro e sucesso.

---

## 🛠️ Tecnologias
- HTML5  
- CSS3  
- JavaScript  

---

## 📊 Resultado
Aplicação funcional que permite ao usuário gerenciar suas tarefas em tempo real, com feedback visual e botões de ação intuitivos.
